# Issam Taki – Web Developer Portfolio

This repository contains the source code for my personal web developer portfolio, showcasing my skills, projects, and technical abilities.

---

## 🚀 Project Overview

**Portfolio Website** is a dynamic web application built to present my profile, projects, and skills as a full-stack web developer.  
It demonstrates my ability to create modern, responsive, and interactive web applications using Laravel and front-end technologies.

---

## 🛠️ Tech Stack

- **Backend:** Laravel (PHP)
- **Frontend:** HTML, CSS, jQuery
- **Styling:** Pure CSS
- **Other:** Responsive design, dynamic content management
- **Database:** (Optional: SQLite / MySQL if needed for contact forms)

---

## 📂 Features

- Dynamic project showcase with multilingual support (EN / FR)
- Skills section displaying frontend and backend expertise
- Contact section (form handling)
- Responsive design for mobile, tablet, and desktop
- Smooth interactions using jQuery
